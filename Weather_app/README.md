installation:

(for linux it might be different)
open up cmd  and write these:

pip install -r requirements.txt

also you need to get your own api-key from https://openweathermap.org/api

first sign-up and then get to your api-key section via your profile picture

activate it and copy-paste it to your "api-key" section in main.py

This Program made with Python 3.11.0 version.
